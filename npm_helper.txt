

всё в папку server

express

pg, pg-hstore - система управления базами данных

sequelize - инструмент для организации взаимодействия
 между платформой Node.js и реляционными базами данными
 без использования специального языка запросов SQL

cors - чтобы обращаться с браузера к серверу

dotenv - даёт возмжность задавать переменные окружения

-D nodemon - чтобы не перезапускать каждый раз сервер

express-fileupload - возможность работать с файлами (загрузка, выгрузка и т д)

uuid - генерация уникального имени
